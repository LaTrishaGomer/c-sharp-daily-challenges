# 10 — Serialization, File I/O & System Utilities

This section focuses on interacting with the filesystem, working with system APIs, and serializing data for storage or transport in C#.  
The exercises emphasize safe file access, structured data handling, and practical system-level operations commonly used in real applications.

---

## Learning Goals

- Read and write files safely
- Serialize and deserialize data
- Work with file paths and system information
- Handle external resources responsibly
- Understand data persistence concepts
