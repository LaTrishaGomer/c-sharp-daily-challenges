# 03 — Algorithms & Problem Solving

This section focuses on implementing fundamental algorithms and structured problem-solving techniques using C#.  
The exercises emphasize logical reasoning, recursion, searching, and algorithmic thinking — core skills for writing efficient and maintainable code.

---

## Learning Goals

- Practice algorithmic thinking and decomposition
- Understand recursion and iterative solutions
- Implement search and sorting logic
- Analyze problem correctness and edge cases
- Strengthen reasoning through structured challenges
