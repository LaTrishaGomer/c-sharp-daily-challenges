# 09 — Delegates, Events & Functional Patterns

This section focuses on decoupling logic using delegates, events, and functional programming patterns in C#.  
The exercises emphasize passing behavior as data, reacting to state changes, and building flexible, event-driven systems.

---

## Learning Goals

- Understand delegates and function references
- Use `Func` and `Action` effectively
- Implement event-driven behavior
- Decouple components through callbacks
- Model reactive program flows
