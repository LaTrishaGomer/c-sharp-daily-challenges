# 05 — Object-Oriented Design

This section focuses on object-oriented programming concepts in C#, including class design, encapsulation, inheritance, abstraction, and common design patterns.  
The exercises emphasize modeling real-world behavior through well-structured types and reusable components.

---

## Learning Goals

- Design classes with clear responsibilities
- Apply encapsulation and access modifiers
- Understand inheritance and polymorphism
- Use constructors and object initialization
- Implement operator overloading and indexers
- Practice common object-oriented patterns
