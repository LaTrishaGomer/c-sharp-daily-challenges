# 04 — Dates, Times & Numeric Computation

This section focuses on working with date and time values, numeric precision, and safe arithmetic operations in C#.  
The exercises emphasize accurate calculations, parsing, and handling edge cases commonly encountered in real-world applications involving time ranges and numerical data.

---

## Learning Goals

- Work with `DateTime` and time calculations
- Parse and validate date inputs
- Handle numeric overflow safely
- Understand differences between numeric types
- Perform precise financial and weighted calculations
