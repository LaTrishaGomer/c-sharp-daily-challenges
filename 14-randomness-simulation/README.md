# 14 — Randomness & Simulation

This section focuses on generating randomized data and simulating behavior using C#.  
The exercises emphasize controlled randomness, data generation, and simple simulation techniques useful for testing, demos, and game-like scenarios.

---

## Learning Goals

- Generate random values safely
- Simulate realistic data behavior
- Understand pseudo-random number generation
- Build simple simulation logic
- Practice reproducible randomness patterns
