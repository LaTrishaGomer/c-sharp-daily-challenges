# 02 — Collections & LINQ

This section focuses on working with collections such as lists, arrays, dictionaries, and queues using core C# features and LINQ.  
The exercises emphasize filtering, transforming, aggregating, and organizing data — skills commonly used when processing real application data.

---

## Learning Goals

- Work confidently with collections (`List`, `Array`, `Dictionary`, `Queue`)
- Understand LINQ querying and transformations
- Filter and project data sets
- Aggregate and group values
- Apply collection-based problem solving
