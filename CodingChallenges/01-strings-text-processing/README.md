# 01 — Strings & Text Processing

This section focuses on manipulating, analyzing, and formatting text using core C# language features.  
The exercises emphasize practical string operations commonly used in real-world applications such as parsing input, formatting output, and transforming textual data.

---

## Learning Goals

- Work confidently with `string` operations
- Practice parsing and formatting text
- Understand character iteration and transformation
- Use `StringBuilder` when appropriate
- Improve problem decomposition using text-based challenges
