# 08 — Reflection, Attributes & Metadata

This section focuses on inspecting types at runtime and using metadata to influence program behavior in C#.  
The exercises emphasize reflection APIs, custom attributes, and attribute-driven design patterns commonly used in frameworks and tooling.

---

## Learning Goals

- Inspect types and members using reflection
- Work with runtime metadata
- Create and apply custom attributes
- Detect behaviors dynamically
- Understand attribute-driven programming patterns
