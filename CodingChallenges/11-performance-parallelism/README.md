# 11 — Performance, Memory & Parallelism

This section focuses on measuring performance, improving memory efficiency, and executing work concurrently in C#.  
The exercises emphasize understanding runtime behavior, optimizing execution, and safely leveraging parallel processing features.

---

## Learning Goals

- Measure execution performance
- Understand memory-efficient data handling
- Use spans for safer memory access
- Apply lazy initialization
- Execute work concurrently and in parallel
