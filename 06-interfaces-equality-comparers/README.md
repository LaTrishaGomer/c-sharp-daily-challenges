# 06 — Interfaces, Equality & Comparers

This section focuses on interface-driven design and defining equality and ordering behavior for custom types in C#.  
The exercises emphasize how objects are compared, sorted, and evaluated inside collections — an essential concept for building predictable and maintainable systems.

---

## Learning Goals

- Implement interfaces to define behavior contracts
- Understand object equality semantics
- Create custom sorting logic
- Implement comparers for collections
- Control ordering and deduplication behavior
